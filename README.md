# 🗂️ DDT79 Territorial Data Platform  
Plateforme de données territoriales — Portfolio Data Analyst (DDT79)

📊 Cette plateforme regroupe l’ensemble des traitements nécessaires pour collecter, nettoyer, structurer, analyser et visualiser des données territoriales.  
Elle démontre mes compétences en tant que Data Analyst dans un contexte DDT (Direction Départementale des Territoires).

---

## 🎯 Objectifs

- Concevoir un **SGBD territorial complet** (PostgreSQL + PostGIS)  
- Collecter, nettoyer et fiabiliser des **données publiques**  
- Produire des **analyses territoriales** utiles aux politiques publiques  
- Générer des **cartes, graphiques et rapports automatisés**  
- Documenter chaque étape de manière professionnelle  
- Centraliser l’ensemble dans une **plateforme cohérente**  

---

## 🗂️ Données publiques utilisées

- **INSEE** — démographie, logement, territoire  
- **DVF** — transactions immobilières  
- **BAN** — adresses  
- **IGN** — référentiels géographiques  
- **SIRENE** — entreprises et établissements  
- **Data.gouv** — jeux de données nationaux  
- **Cadastre** — parcelles, bâti  
- **OpenStreetMap** — données géographiques libres
